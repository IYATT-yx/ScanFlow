# ScanFlow 扫码流水线

2026/7/23  
某款产品因为需要记录实际包装发货件的追溯码，于是有了这个软件。  
每件产品上标刻有追溯二维码，配合扫码枪就可以录入。  
![alt text](docs/image1.png)  

## 打包

* Python 3.8.10 32位（支持 Windows 7 及以上）  
最开始我用的 64 位，拿到现场去测试才发现配的工控电脑是 32 位的 Windows 7，用 32 位 Python 可以保证同时在 32 位和 64 位的电脑上正常运行。
